# 🧮 SGPA Formula

```
SGPA = Σ(Credit × Grade Point)
       -----------------------
          Total Credits
```

Example:

| Subject | Credits | Grade | Grade Point | Earned |
|----------|---------|-------|------------:|--------:|
| Computer Networks | 3 | A+ | 10 | 30 |
| DAA | 4 | A | 9 | 36 |
| Web Technologies | 4 | B+ | 8 | 32 |

Final SGPA

```
Total Grade Points / Total Credits
```

---

# 📊 CGPA Formula

```
CGPA = Sum of all Semester SGPAs
       -------------------------
        Number of Semesters
```

Example

Semester 1 → 8.20

Semester 2 → 8.55

Semester 3 → 8.90

Semester 4 → 8.74

Semester 5 → 9.15

```
CGPA = (8.20 + 8.55 + 8.90 + 8.74 + 9.15) / 5
```
