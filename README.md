# 🧮 SGPA Formula

```
SGPA = Σ(Credit × Grade Point)
       -----------------------
         Total Credits
```

Example

```
Computer Networks

Credits = 3

Grade = A+

Points = 10

Earned = 3 × 10 = 30
```

All subjects are calculated similarly.

---

# 📊 CGPA Formula

```
CGPA = Sum of Semester SGPAs
       ----------------------
       Number of Semesters
```

Example

```
Semester 1 = 8.10
Semester 2 = 8.72
Semester 3 = 9.01
Semester 4 = 8.65
Semester 5 = 9.12

CGPA = (8.10 + 8.72 + 9.01 + 8.65 + 9.12) / 5
```
